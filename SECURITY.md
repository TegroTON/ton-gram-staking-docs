# Security policy

## Reporting a vulnerability

If you discover a security issue in the Tegro staking protocol, contracts, or app, please report it
**privately** and give us a reasonable chance to remediate before any public disclosure.

- **Telegram:** [t.me/TegroFinance](https://t.me/TegroFinance)
- Do **not** open a public GitHub issue or post exploit details publicly.

Please include: a clear description, affected component/address, reproduction steps or proof of
concept, and impact. We appreciate responsible disclosure.

## Scope

This repository is documentation only. Reports may concern:

- the stgTON staking contracts on TON mainnet (see [docs/contracts.md](docs/contracts.md)),
- the public staking API (`https://tegro.finance/api/v1`),
- the app at `https://tegro.finance/staking`.
