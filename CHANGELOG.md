## [1.0.0] — 2026-06-09

Initial public release of the Tegro liquid staking documentation.

### Added
- **Docs:** how it works, architecture (diagrams), step-by-step guide, FAQ, contract registry,
  security model, API reference, AI/agent integration, disclaimer.
- **Examples:** runnable TypeScript, Python, curl, and direct on-chain reads.
- **Machine-readable:** `llms.txt` index and `openapi.yaml` (OpenAPI 3.0) for the staking API.
- **Project files:** README, CONTRIBUTING, SECURITY, LICENSE (MIT / CC BY 4.0), issue templates.
